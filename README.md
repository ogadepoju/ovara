# 🌸 PCOS Home Risk Check

A friendly, plain-English PCOS risk questionnaire for home users.

## Deploy to Render (step by step)

### Step 1 — Train the model locally (once only)
```bash
pip install -r requirements.txt
pip install kagglehub  # only needed for this step

# Set your Kaggle credentials
# Windows PowerShell:
$env:KAGGLE_USERNAME="your_username"
$env:KAGGLE_KEY="your_api_key"

python train.py
# → generates trained_model.pkl (~5MB)
```

### Step 2 — Push to GitHub
```bash
git init
git add .
git commit -m "initial deploy with pre-trained model"
git remote add origin https://github.com/YOUR_USERNAME/pcos-home.git
git push -u origin main
```

### Step 3 — Create Render service
1. Go to render.com → New → Web Service
2. Connect your GitHub repo
3. Render auto-detects render.yaml
4. No environment variables needed (model is bundled)
5. Click Deploy

Your app will be live at: https://pcos-home.onrender.com

## Run Locally
```bash
pip install -r requirements.txt
python train.py          # first time only
python -m uvicorn main:app --reload --port 8000
```

## Project Structure
```
pcos_home/

├── train.py             ← Run locally once to generate model
├── trained_model.pkl    ← Generated by train.py, committed to git
├── templates/
│   └── index.html       ← Questionnaire UI
├── static/
├── render.yaml          ← Render deployment config
├── requirements.txt     ← No kagglehub needed on server
├── .gitignore
└── README.md
```
